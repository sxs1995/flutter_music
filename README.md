# fultter_music

实现音乐播放器的基本布局,调用 QQ 音乐接口渲染页面

## 首页拆分为轮播和歌单部分

### 歌单部分采用两个方式

1.musicList.1.dart 中采用 Future 的方式
2.musicList.dart 中采用创建数据模型的方式

### 插件

1.flutter_screenutil: ^0.5.1 屏幕适配
2.flutter_swiper: ^1.1.5 轮播
3.azlistview: ^0.1.0 城市字母选择
